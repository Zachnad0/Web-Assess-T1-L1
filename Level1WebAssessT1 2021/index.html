How I made this strange website (part 1)

There are multiple CSS animations used in this "website". How you make an animation is by first naming the animation in your stylesheet under the selector of the element you wish to animate. the syntax goes like so: animation-name:[insert name here];
Remember, never include the square brackets, they just show you what is interchangeable. Next, you are going to want to set a total duration using this CSS tag:
animation-duration:[however many seconds you want][s for seconds or ms for milliseconds]; Now set a direction. The different modes are self explanatory, except for alternate, which goes forwards and then backwards constantly in a loop for as long as another tag in the seconds part specifies:
animation-direction:[normal/reverse/alternate/reverse alternate]; GO TO PART 2 -->

How I made this strange website (part 2)

Read the first part now if you have not yet. Now you may want to set the fill mode. The fill mode is only usefull if you don't want the animation to play on loop by freezing it on the first or last frame like so: animation-fill-mode:[forwards/backwards]; There is one more very important tag that you may want for your animation if you want to loop it, it is the:
animation-iteration-count:[amount/infinite]; change amount to be a number of however many times you want it to loop, if at all, or put in infinite for it to loop, well, infinitely. Now for the keyframes. keyframes go from 0% to 100%, and transition between each state at the necessary speed needed to finish the animation in the time set as the animation duration. To specify and animate, GO TO PART 3 (bottom right corner).

How I made this strange website (part 4)

Read the first part now if you have not yet. Here I am going to show you two examples of animations I have used on the website. Here is the animation for the animated central div container:
@keyframes centerpieceBackground { from { width: 40%; height:28%; top: 36%; left: 29%; } to { width: 60%; height:48%; top: 26%; left: 19%; } }
Here are the css animation controls: animation-name: centerpieceBackground; animation-duration: 2s; animation-play-state: running; animation-direction: alternate; animation-iteration-count: infinite; animation-delay: 0.5s;
Here is another animation from my site that controls the background:
@keyframes strobe-bg { 0% {background-color: blue;} 25% {background-color: purple;} 50% {background-color:orange;} 75% {background-color:green;} 100% {background-color: red;} }
And here are the CSS controls: animation-name: strobe-bg; animation-iteration-count: infinite; animation-duration: 3s; animation-direction: alternate;

How I made this strange website (part 3)

Read the first part now if you have not yet. Now to make some keyframes. Start off by making a keyrfame in your stylesheet like so: @keyframes [animation name] {[animations in here]}
Replace the first square brackets section with the name of the animation which you have specified already. To program the animation, you must specifiy the starting ending position, or enter percentages to specifiy at what percentage of the time through the animation will it do something. Here is an example of a simple to and from animation keyframe setup that changes the background color from blue to green:
@keyframes example {from{background-color:blue;} to{background-color:green;}}
Now here is one that has 4 steps increasing by (100%/4 == 25% per step): @keyframes example {0%{color:red;} 25%{color:green;} 50%{color:blue;} 75%{color:yelllow;}}
